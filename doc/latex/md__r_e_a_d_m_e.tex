HyperC is an implementation of the \href{CCSDS_Compression.pdf}{\texttt{ CCSDS-\/123 B-\/2}}, designed to meet the FINCH-\/\+Payload\+Controller-\/\+Lossless\+Compression requirement. The required Compression Speed is givenby the \href{https://www.notion.so/utat-ss/FINCH-PayloadController-CompressionSpeed-0a9a3c2992e44a239bd1d1e7e464b6c9}{\texttt{ FINCH-\/\+Payload\+Controller-\/\+Compression\+Speed}} requirement. The compression rate required is provided by the \href{https://www.notion.so/utat-ss/FINCH-PayloadController-CompressionRate-8cabbe2393b147b6ae5e73e8ee58c9f4}{\texttt{ FINCH-\/\+Payload\+Controller-\/\+Compression\+Rate}} requirement.

This project contains\+:


\begin{DoxyItemize}
\item Compressor
\item Decompressor
\item Hyperspectral Image formatting and generation
\end{DoxyItemize}

The raw hyperspectral image files are expected to be .uhi (Uncompressed Hyperspectral Image) files, which makes getting and storing individual pixels much faster than csv or mat files.

Compressed Images are stored in .vuf files (Variable Unsigned Files), which are capable of storing unsigned integers of any length less than 32 bits.\hypertarget{md__r_e_a_d_m_e_autotoc_md3}{}\doxysection{\texorpdfstring{Installation Instructions}{Installation Instructions}}\label{md__r_e_a_d_m_e_autotoc_md3}

\begin{DoxyCode}{0}
\DoxyCodeLine{git\ clone\ https://github.com/Kevin-\/Caldwell/CCSDS-\/Compression-\/C.git}

\end{DoxyCode}

hc-\/compress is the Image Compressor, and the CLI Tags are\+:


\begin{DoxyItemize}
\item {\ttfamily -\/s} \+: Set Source File
\item {\ttfamily -\/d} \+: Set Destionation File
\item {\ttfamily -\/a} \+: Set Action (See Below)
\item {\ttfamily -\/D} \+: Enable {\ttfamily DEBUG MODE}
\item {\ttfamily -\/l} \+: Set Log File
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md7}{}\doxysubsubsection{\texorpdfstring{Actions}{Actions}}\label{md__r_e_a_d_m_e_autotoc_md7}
Actions describe all the functions you can execute using hc-\/compress. The list of possible actions are\+:


\begin{DoxyItemize}
\item {\ttfamily compress}\+: Compress an image in a .uhi file to a .vuf file
\item {\ttfamily predict}\+: Run the predictor on an image. Input and output files are .uhi
\item {\ttfamily encode}\+: Run Encoder on an image in a .uhi file into a .vuf file
\item {\ttfamily test-\/x}\+: Validate x (predictor, encoder, compressor, uhi, vuf)
\end{DoxyItemize}
